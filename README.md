# washeprawo

## Overview

Тестовое приложение для washeprawo

## Start

```
$ git clone https://github.com/Babrakov/washeprawo.git
$ cd washeprawo
```
Загрузить дамп из migrate.sql, настроить параметры подключения к БД в settings.php
